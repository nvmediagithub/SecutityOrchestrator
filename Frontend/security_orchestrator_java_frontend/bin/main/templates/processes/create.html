File Format Guidelines
description BPMN Files
  • Standard BPMN 2.0 XML format
  • File extensions: .bpmn, .xml

api OpenAPI Specifications
  • OpenAPI 3.0+ JSON or YAML format
  • File extensions: .json, .yaml, .yml
  • Should define REST API endpoints
